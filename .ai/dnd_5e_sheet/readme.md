# Modulo: Ficha D&D 5e (Product)

## 1. Proposito
Definir as regras especificas futuras para fichas de personagens do sistema D&D 5e.

No MVP, este modulo existe como fronteira de produto e arquitetura. A implementacao inicial trata `DND_5E` apenas como valor de compatibilidade entre personagem, ficha e campanha.

## 2. Escopo Futuro
* Campos mecanicos de D&D 5e.
* Validacao de atributos, classe, raca, background, nivel e recursos.
* Calculo de modificadores, proficiencia, defesa, iniciativa e HP.
* Regras de magias, equipamentos e pericias.

## 3. Fora de Escopo no MVP
* Implementar calculos completos.
* Criar interface completa de ficha.
* Criar ou vincular personagens a campanhas.
