# Modulo: Ficha Pathfinder 2e (Product)

## 1. Proposito
Definir as regras especificas futuras para fichas de personagens do sistema Pathfinder 2e.

No MVP, este modulo existe como fronteira de produto e arquitetura. A implementacao inicial trata `PATHFINDER_2E` apenas como valor de compatibilidade entre personagem, ficha e campanha.

## 2. Escopo Futuro
* Campos mecanicos de Pathfinder 2e.
* Validacao de ancestralidade, background, classe, nivel, proficiencias e recursos.
* Calculo de bonus, DCs, defesa, iniciativa e HP.
* Regras de talentos, magias, equipamentos e acoes.

## 3. Fora de Escopo no MVP
* Implementar calculos completos.
* Criar interface completa de ficha.
* Criar ou vincular personagens a campanhas.
