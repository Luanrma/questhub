# Modulo: Ficha Pathfinder 2e (Skills & Tech)

## 1. Stack
* TypeScript para validadores e calculos.
* Zod para contratos de entrada.

## 2. Padroes
* Strategy por sistema.
* Value Objects para valores mecanicos.
* Testes unitarios para regras e calculos.

## 3. Restricoes
* Nao criar nem vincular `Character`.
* Nao acessar canvas/tabuleiro diretamente.
* Expor validadores consumidos por `character_sheet`.
